### Retail Store Sales Analysis in Excel
This repository contains a complete Excel retail analytics project created from the Retail-Store-Transactions.xlsx dataset.

## Project objective
The project analyzes sales performance across stores, products, payment methods, cashiers, managers, and time-based trends.

Dataset columns
Date
Time
StoreID
Location
Product
Quantity
UnitPrice
PaymentType
TransactionID
Cashier
StoreManager
TimeOfDay
DayOfWeek
TotalPrice

# Final workbook contents
Dashboard - KPI cards and charts for executive summary
Raw_Data - cleaned transaction-level data with helper fields
Overview - overall project metrics
Store_Performance - store and location analysis
Product_Performance - product revenue and quantity analysis
Payment_Analysis - payment type revenue contribution
Day_Time_Analysis - weekday and time-of-day sales matrix
Monthly_Trend - monthly revenue trend
Cashier_Performance - cashier-level productivity analysis
Manager_Performance - manager-level performance analysis

# Features added
Duplicate removal check using TransactionID
Revenue validation using Quantity * UnitPrice
Helper columns: ComputedTotal, TotalCheck, Year, Month, Hour
Formatted Excel tables in every sheet
Ready-to-show dashboard charts inside Excel
